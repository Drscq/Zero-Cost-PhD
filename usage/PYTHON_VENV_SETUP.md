# Python Virtual Environment Setup

## Create
```bash
python -m venv venv
```

## Activate (macOS / Linux: bash, zsh)
```bash
source venv/bin/activate
```

## Deactivate
```bash
deactivate
```

## Upgrade pip (optional)
```bash
python -m pip install --upgrade pip
```

## Verify
```bash
python -V
pip list
```
